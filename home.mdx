## About Us
ERLCHelper is a multi-purpose bot for all of your ERLC (Emergency Response: Liberty County) needs. We offer several API-based features, ranging from moderation tools to growing your community. We are NOT affiliated with ERLC or PRC - we are a third party service using the API provided by them. With the ERLC scene growing bigger and bigger everyday, having a multi-purpose tool designed for your server is very important! ERLCHelper is a trustworthy and safe tool that promotes growth, efficiency, and satisfaction. We provide several amazing features at no cost! 

## Documentation
This documentation is a great resource for users who have just started to use ERLCHelper. This documentation provides useful guides on how to configure ERLCHelper to ensure a smooth experience when using our bot. If at anytime you have a question about ERLCHelper please do not hesitate to contact our support team [here](https://discord.gg/cPKT6prP7q).
